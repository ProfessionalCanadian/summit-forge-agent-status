---

## Art Style Options

### Option A — Stylized 3D Render
Think Pixar-meets-cyberpunk: characters rendered in 3D with cinematic lighting, clean textures, expressive faces. Premium and modern. High production feel. Each agent looks like a character from an animated feature.

### Option B — Painted Illustration (Digital Oil/Ink)
Rich, textured portraits — each agent as a hand-painted character with painterly detail. Feels exclusive, gallery-quality. Conveys craftsmanship. Best for showing personality and mood states.

### Option C — Flat Vector / Icon Art
Clean, modern, designer-grade vector characters. Simple geometry, bold color blocking, no noise. Scales perfectly at any resolution. Best if we want the UI to feel like a premium SaaS product.

---

## Style Notes from Oscar

My lean: **Option B** — painted illustration. "Summit Forge" evokes craftsmanship and mastery. A painted portrait style makes each agent feel like a named character with weight and identity. States (sleeping, working, degraded) are naturally expressive in painted form.

That said — this is a team decision.
